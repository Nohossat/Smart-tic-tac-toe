# Tic-Tac-Toe

2 modes available:

### Human versus Human

In the app.py file, change the human parameter to **True**.

```python
morpion_game = Morpion(human=True)
```

### Human versus Computer

In the app.py file, change the human parameter to **False**.

```python
morpion_game = Morpion(human=False)
```
